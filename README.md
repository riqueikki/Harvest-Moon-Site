# 🌾 Harvest Moon SNES - Guia do Fazendeiro

Este é um projeto de site enciclopédico sobre o clássico jogo **Harvest Moon** para Super Nintendo (SNES). O objetivo é fornecer informações detalhadas sobre os habitantes do vilarejo, criação de animais e mecânicas do jogo com uma estética retrô fiel aos 16-bits.

## 🚀 Funcionalidades

- **Galeria de Personagens:** Fichas completas com gostos e localizações de Jack, as pretendentes (Ellen, Nina, Maria, Anne, Eve) e NPCs importantes (Prefeito, Padre, Carpinteiro).
- **Criação de Animais:** Informações sobre Cavalos, Vacas, Galinhas e o fiel Cachorro.
- **Design Retrô:** Interface inspirada na identidade visual do jogo, utilizando fontes pixeladas e cores clássicas.
- **Navegação Dinâmica:** Sistema de cards interativos para fácil acesso às informações.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais para garantir leveza e compatibilidade:

* **HTML5:** Estruturação semântica de todas as páginas e fichas técnicas.
* **CSS3:** Estilização personalizada, incluindo:
    * Layout em Grid e Flexbox para a galeria de personagens.
    * Efeitos de hover nos cards de seleção.
    * Renderização de imagem pixelada (`image-rendering: pixelated`) para preservar a arte do SNES.
* **Git & GitHub:** Controle de versão e hospedagem do código.
